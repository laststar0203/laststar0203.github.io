공부 정리 작성 프로세스

공부 -> 배운 내용을 노트에 정리 -> github 정리

공부후 배운 내용을 노트에 정리는 항상 이루어지나 배운 내용을 노트의 정리 -> github 정리는 아니다.

실수한 점
1. 한 문단 끝날때마다 정리하지 않기
    : 시간이 너무 걸림
-> 공부 시간, 정리 시간을 정해놓은 다음 공부시간 동안 최대로 할수 있는 만큼 해놓은다 그 시간이 끝나면 정리 시간에 지금 까지 공부한 내용을
한 문단 단위로 정리한다 단위가 되지 않은 부분은 삭제한다. 나중에 다시 공부하여 채우도록 한다.

왜냐면 한 문단이 독립적인 내용을 가진게 아니라 여러 문단이 서로 연결된 관계를 가질 수 있기 때문에 이를 고려하여 정리하는 문단을 잘 나눌 수 있도록 하기 위해서이다.



정리할때 궁금한점들도 적어놓아도 나쁘지 않을것 같다.
